## 优劣与使用场景

很多项目管理软件都支持了甘特图，我究竟是挑选那选重量级软件，还是ganMaker呢？

下面是一个列表：

| #        | ganMaker                      | 大型项目管理软件                   |
| -------- | ----------------------------- | ---------------------------------- |
| 优点     | 简单，快速                    | 时间控制跟紧缺，有的还可以分配任务 |
| 适用场景 | 个人报告，ppt中，主要用来展示 | 实际项目中，主要用来供内部控制进度 |
